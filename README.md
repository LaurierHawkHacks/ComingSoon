### Requirements
- Node.js, PNPM

### Setup
- Clone repo
```sh
$ git clone https://github.com/LaurierHawkHacks/ComingSoon
```
- Install dependencies 
```sh
$ pnpm i
```
- Launch local server
```sh
$ pnpm dev
```
